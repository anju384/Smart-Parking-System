# Smart Parking Management System

A software-based indoor parking system that allows users to reserve parking
slots using a QR-based interface and simulates automated entry and exit.

This was developed as a course-based team project to understand how real-world
automation systems can be modeled using software.

## Technologies
- Python
- Flask
- QRCode
- SQLite
- HTML/CSS

## Features
- Slot selection via web interface
- QR-based exit verification
- Real-time slot status updates
- Modular design for future AI integration

## Run

pip install -r requirements.txt  
python app.py  

Open: http://127.0.0.1:5000
